# Hospital Landing Page

Modern healthcare landing page built with Gatsby.js and Tailwind CSS.

## Features

- Responsive medical UI
- Dynamic reviews API
- Modern animations
- Mobile navbar
- Professional design
- Vercel deployment

## Tech Stack

- Gatsby.js
- Tailwind CSS
- Axios
- Framer Motion
- React Icons

## API

https://admin.tomedes.com/api/v1/get-reviews?page=1

## Installation

npm install
gatsby develop

## Deployment

Hosted on Vercel
